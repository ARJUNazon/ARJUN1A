# AZON Green Zone

AZON Green Zone is a responsive restaurant website built using HTML, CSS, and JavaScript.

## Features
- 🍽️ Responsive Design
- 🔍 Search Functionality
- 🛒 Shopping Cart
- 🌗 Dark/Light Theme
- 🥗 Veg & Non-Veg Menu Filter
- 🖼️ Food Gallery
- 📞 Contact Form
- 📱 Mobile Friendly

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages
